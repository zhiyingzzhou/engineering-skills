# 页面类型与复刻重点

## 营销站 / Landing

重点：

- hero / masthead
- 品牌条
- 功能段
- 定价
- FAQ
- CTA / footer

## 内容站 / 博客 / 文档站

重点：

- 列表页
- 详情页排版节奏
- 目录
- 代码块
- 标签 / 分页 / 阅读导航

## Dashboard / 应用站

重点：

- 导航壳层
- table / filter / tab / KPI 卡
- dropdown / modal / loading / empty
- 状态色

## 电商 / 目录站

重点：

- 商品卡
- 图片比例
- 筛选 / 排序
- 价格与促销态
- 详情页表单

## 认证 / 表单流程

重点：

- 输入框
- focus / error / disabled
- 社交登录按钮
- 密码切换
- 次级操作链接
